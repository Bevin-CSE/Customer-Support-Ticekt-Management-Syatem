# Customer Support Ticket Management System

This project is a **Customer Support Ticket Management System** developed to manage and track customer issues efficiently. It uses **Azure DevOps Boards** for planning (Epics, Features, User Stories), along with system design diagrams like Architecture, ER, Class, Sequence, and Use Case diagrams.

The project also includes **load testing, sprint planning, test cases, and pipeline setup**, demonstrating a complete DevOps workflow. The implementation is supported by frontend code (HTML, CSS, JavaScript) and organized using GitHub for version control.

Overall, this project showcases a structured approach to software development, from planning and design to testing and deployment.
